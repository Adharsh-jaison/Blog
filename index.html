How to Create an Account in Google Tag Manager

  1. Go to the Google Tag Manager website: Open your web browser and navigate to tagmanager.google.com.
  2. Sign in with your Google account: If you’re not already signed in, you’ll be prompted to log in using your Google credentials.
  3. Create a new account: Click on the “Create Account” button. Fill in your account name (usually your company name), and select your country.
  4. Set up a container: After creating your account, you’ll need to set up a container. A container holds all your tags. Give it a name (e.g., your website’s name), select where you’ll use the container (Web, iOS, Android, AMP), and click “Create.”
  5. Accept the terms of service: Review and accept the Google Tag Manager Terms of Service Agreement.
  6. Install the container code on your website: After creating your container, you’ll be provided with a snippet of code. Copy this code and paste it into your website’s code, typically before the closing </head> tag.
  7. Start using Google Tag Manager: Once the code is added to your site, you can start adding and managing tags through the Google Tag Manager interface.

How to Create an Account in Google Analytics 4

  1. Go to the Google Analytics website: Open your web browser and navigate to analytics.google.com.
  2. Sign in with your Google account: If you’re not already signed in, you’ll be prompted to log in using your Google credentials.
  3. Create a new account: Click on the “Start measuring” button to create a new Google Analytics account. Enter your account name, configure your data sharing settings, and click “Next.”
  4. Set up a property: A property represents your website or app. Enter a name for your property, select your reporting time zone, and currency. Click “Next.”
  5. Set up data streams: Choose the platform you want to measure (Web, iOS, Android), and set up a data stream by entering your website URL and stream name.
  6. Install the tracking code: Google Analytics will provide you with a tracking code. Copy the code and paste it into your website’s code, typically before the closing </head> tag.
  7. Start using Google Analytics 4: Once the code is added to your site, Google Analytics will begin collecting data, which you can view in your GA4 dashboard.
